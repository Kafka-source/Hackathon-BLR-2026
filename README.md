# Pixel - Microsoft Teams IT Support Bot

## Overview

**Pixel** is an AI-powered Microsoft Teams bot that helps users troubleshoot IT issues through natural conversations. Users can add **Pixel** as a Teams app and chat with it just like any other contact.

Powered by **NeuroSAN by Cognizant**, Pixel provides intelligent troubleshooting assistance and integrates with **ServiceNow REST APIs** to create and update IT support tickets when required.

## Features

- AI-powered IT troubleshooting
- Installable as a Microsoft Teams app
- Create ServiceNow incidents
- Update existing ServiceNow incidents
- Powered by NeuroSAN by Cognizant

## Technology Stack

- Microsoft Teams API
- Azure Bot Service
- NeuroSAN by Cognizant
- ServiceNow REST APIs
- Microsoft Teams Developer Portal

## How It Works

1. Add **Pixel** as an app in Microsoft Teams.
2. Start chatting with Pixel about your IT issue.
3. Pixel provides troubleshooting guidance using NeuroSAN.
4. If needed, Pixel creates or updates a ServiceNow ticket and shares the incident details.

## Bot Management

- **Hosting:** Azure Bot Service
- **Teams App Management:** Microsoft Teams Developer Portal
- **ITSM Integration:** ServiceNow REST APIs
